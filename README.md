# 3. Essential AI Terminology

## Artificial Intelligence (AI)

Machines performing tasks that normally require human intelligence.

Examples:

-   language understanding
-   image recognition
-   speech recognition

------------------------------------------------------------------------

## Machine Learning

Systems that learn patterns from data.

Example:

Spam email classification.

------------------------------------------------------------------------

## Deep Learning

Neural networks with multiple layers used for complex tasks.

Used in:

-   computer vision
-   speech recognition
-   natural language processing

------------------------------------------------------------------------

## Transformers

Architecture powering modern LLMs.

Popular models:

-   GPT
-   Claude
-   Gemini
-   LLaMA

------------------------------------------------------------------------

## Large Language Models (LLMs)

Models trained on massive text datasets.

Example:

"The capital of France is" → Paris

------------------------------------------------------------------------

## Prompt Engineering

Designing instructions for LLMs to get useful responses.

Example

Bad prompt

Explain AI

Better prompt

Explain AI to a beginner with simple real-world examples.

------------------------------------------------------------------------

## Embeddings

Numeric representations of text used for semantic similarity.

Used in:

-   search
-   recommendation systems
-   vector databases

------------------------------------------------------------------------

## Retrieval Augmented Generation (RAG)

Instead of training a model with private data, RAG retrieves relevant
documents and sends them to the LLM.

Pipeline

User Question\
↓\
Vector Search\
↓\
Relevant Documents\
↓\
LLM Response

------------------------------------------------------------------------

## AI Agents

AI systems capable of:

• reasoning\
• planning\
• tool usage\
• executing tasks

Example

User: Research electric vehicles

Agent workflow:

Search web → analyze information → summarize findings
